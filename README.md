## 2. Supervised Learning (Regression & Classification)

* **[Linear Regression: E-commerce Customer Spending](https://github.com/agrim-git-hub/Linear-Regression-Ecommerce-Customers)**
    > Using Linear Regression to predict the "Yearly Amount Spent" by e-commerce customers based on their app/website usage and membership length.

* **[Logistic Regression: Titanic Survival Prediction](https://github.com/agrim-git-hub/Logistic-Regression-Titanic-Survival)**
    > Using Logistic Regression to predict passenger survival on the Titanic based on features like age, class, and gender.

* **[K-Nearest Neighbors Project](https://github.com/agrim-git-hub/K-Nearest-Neighbors-Project)**
    > Applying the K-Nearest Neighbors (KNN) algorithm to a classified dataset to predict a target class, including a step to find the optimal 'K' value.

* **[Decision Trees & Random Forest: Loan Prediction](https://github.com/agrim-git-hub/Decision-Forest-Loan-Prediction)**
    > Predicting whether a borrower will pay back their loan in full, comparing the performance of Decision Trees and Random Forest models.

* **[Support Vector Machines: Iris Flower Classification](https://github.com/agrim-git-hub/SVM-Iris-Flower-Classification)**
    > Using Support Vector Machines (SVM) and GridSearch to build a model capable of classifying different species of Iris flowers.
